# Помощник рыбака

Веб-приложение для ведения электронного журнала улова, отслеживания погодных условий и проверки правил рыболовства.

## Приступая к работе

Эти инструкции помогут вам запустить копию проекта на вашем локальном компьютере для целей разработки и тестирования. Инструкции по развертыванию проекта в рабочей среде см. в разделе **Развертывание**.

### Предпосылки

Для работы проекта требуется следующее ПО:

*   Python 3.14
*   Bottle.py

```bash
sudo apt install python3 python3-pip
pip3 install bottle
```

### Установка

Пошаговая серия примеров, которые покажут, как запустить среду разработки:

```
Откройте VSCode
```

```
Запустите файл .py
```

```bash
python3 app.py
```
